# FaceFXKit - iOS Photo Editor

A professional iOS photo editor application focused on facial feature enhancement.

### Filter Categories

#### Size Filters
- Head

#### Hair Filters
- Hair Colors
- Temperature
- Tint
- Saturation
- Other Color Effects

## Requirements

- iOS 26.0+
- Xcode 26.1+

---

*Built with ❤️ using SwiftUI and Core Image*
